Configuration 설정파일 자동생성

개요

전자정부 프레임워크를 사용하여 프로그램을 개발할 때 주로 사용되는 설정 파일(XML) 을 설정파일 마법사를 통하여 자동으로 생성하는 기능이다. 제공하는 설정파일 Template 은 다음과 같다.

사용법

1. Data Source : 데이터베이스 연결을 설정.

  • New DataSource : C3P0, DBCP, JDBC Driver Type의 DataSource 설정.
  • New JNDI DataSource : JNDI Lookup을 이용하여 Database Connection을 생성하는 DataSource 설정 (Weblogic, Jeus 설정 제공)

2. Transaction : 전자정부 프레임워크 Transaction 관리 설정

  • New DataSource Transaction : DataSource를 사용하여 Local Transaction 관리 설정.
  • New JPA Transaction : JPA EntityManagerFactory를 이용한 Transation 관리 설정
  • New JTA Transaction : JTA를 이용한 Global Transation 관리 설정

3. Cache : 전자정부 프레임워크 Cache Service 를 설정.

  • New Cache : 기본 Cache, Custom Cache 설정.
  • New Ehcache Configuration : Ehcache 설정.

4. Id Generation : 시스템을 개발할 때 필요한 유일한 ID를 생성하기 위한 설정.

  • New Sequence ID Generation : Database의 SEQUENCE를 사용하는 서비스 설정.
  • New Table ID Generation : 별도의 테이블을 이용한 ID 생성 서비스 설정
  • New UUID Generation : Universally Unique Identifier(UUID) 생성 서비스 설정

5. Logging : logging 관련 설정 파일 작성

  • New Console Appender : log를 콘솔화면으로 출력하기 위한 appender 설정
  • New File Appender : log를 file로 출력하기 위한 appender 설정
  • New Rolling File Appender : 로그파일이 특정 크기 이상으로 커지지 않도록

백업 기능을 수행하는 RollingFileAppender 설정.

  • New Daily Rolling File Appender : 설정한 날짜 또는 조건에 맞춰 로깅을 수행하는

DailyRollingFileAppender 설정.

  • New JDBC Appender : DB에 로그를 출력하기 위한 Appender 설정.
  • New eGovDB Appender : 전자정부 확장 Appender인 EgovDBAppender 설정 (실행환경 Guide 참조)
  • New eGovJDBC Appender : 전자정부 확장 Appender인 EgovJDBCAppender 설정 (실행환경 Guide 참조)

6. Property

  • New Property : Property Service 설정

7. Scheduling : 실행환경의 Scheduling 서비스 관련 XML 파일 작성

  • New Detail Bean Job : JobDetailBean을 이용한 작업 생성
  • New Method Invoking Job : MethodInvokingJobDetailFactoryBean을 이용한 작업 생성
  • New Simple Trigger : SimpleTriggerBean을 이용한 설정
  • New Cron Trigger : CronTriggerBean을 이용한 설정
  • New Scheduler : SchedulerFactoryBean 을 이용한 스케줄링한 작업의 시작 설정
