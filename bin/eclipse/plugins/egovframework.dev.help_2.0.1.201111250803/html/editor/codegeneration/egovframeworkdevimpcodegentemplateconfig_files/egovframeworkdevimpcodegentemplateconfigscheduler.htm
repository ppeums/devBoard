Scheduler Code Gen

개요

설명

Scheduler Configuration 의 설명은 다음 실행환경 가이드를 참조한다.

  • 실행환경 Scheduling Configuration Guide

사용법

1. Template 목록 조회 : Window>Show View>eGovFrame Templates 를 선택한다.


2. 작성하고자 하는 Configuration의 템플릿을 선택한다.


3. 화면 UI 를 통해 필요한 항목을 입력한 뒤 최종적으로 Finish 버튼을 클릭한다.


4. 생성된 Configuration 파일을 확인한다.
